Tentang Saya


Halo, nama saya Restu. Saya lahir di Banyumas, tetapi saya besar dan bersekolah di Jakarta hingga SMA. Lulus SMA, Saya melanjutkan studi ke Universitas Gadjah Mada, Yogyakarta. Jurusan saya adalah Ilmu Komunikasi. Setelah lulus pada 2018, saat ini saya belajar di Hacktiv8 untuk mewujudkan mimpi saya menjadi developer yang handal. Keseharian saya gunakan untuk belajar sambil sesekali berolahraga yaitu tenis. Saya menyukai berbagai olahraga dan jika ada orang yang ingin menantang saya melakukan "kuis trivia" dalam olahraga apapun, hubungi saja saya melalui kontak yang ada.
